IBorescope Live View

This is a live stream broadcast from the IBorescope device across a Wi-Fi network that is accessed via a Application called Webcam7


The website connects directly to the local server and display's the stream across HTTP and can be accessed by any browser. Currently this is limited to a local machine however this could easily be changed if the website was hosted on a server making it accessible to the Internet.


On a live system, login and authenticaion would be required due to the nature of what the device would be used for and patient confidentiality. Given additional time this would easily be accomplished as the most difficult portion of the project was gaining access to the video steam from the device and doing it securely.
